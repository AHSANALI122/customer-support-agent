# Project: E-commerce Customer Support Agent

Tech stack: FastAPI, SQLModel, LangChain, Gemini API (Google AI Studio),
Chroma, uv, HTML + Tailwind.

Full feature spec lives in `spec.md` (F1–F14, each with requirements and
acceptance criteria). Always read `spec.md` at the start of a session
before writing any code.

Rules:
- Implement only the feature I explicitly ask for in this session.
  Don't jump ahead to later features.
- Match the conventions already established in earlier features
  (e.g. tool parameter names from F4) — don't invent new ones.
- After implementing, list which acceptance criteria from `spec.md`
  are satisfied so I can verify before we move on.
